🚫 元素屏蔽规则

CSS 选择器规则

0 条规则
选择器语法说明：

#id名 → 屏蔽特定 ID 的元素（如 #doubao-ai-assistant）
.class名 → 屏蔽特定 class 的元素（如 .ad-banner）
[attr] → 屏蔽含特定属性的元素（如 [data-ad]）
div.ads → 屏蔽 class="ads" 的 div 元素

提示：支持任意有效的 CSS 选择器

🔍 属性值筛选规则

0 条规则

根据元素属性值进行更复杂的匹配，支持长度比较、正则匹配等操作。

属性规则示例：

style 长度 > 5000 → 移除 style 属性超过 5000 字符的元素
src 包含 utm_ → 移除 src 属性包含 utm_ 的元素
width 数值 > 1000 → 移除 width 属性值超过 1000 的元素

提示：可组合多个条件，选择"满足所有条件"或"满足任一条件"
